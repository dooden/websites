Willkommen!

Webseiten bestehen aus rechteckigen Kästchen. Alles Runde ist entweder Trick, Grafik oder beides.

Beim Umgang mit diesen Kästchen haben HTML und CSS klar getrennte Aufgaben:

  • HTML-Elemente strukturieren die Webseite und erstellen die Kästchen.
  • CSS-Regeln gesatlten die Kästchen und deren Inhalte.

Beide Sprachen arbeiten bei der Erstellung und Gestaltung von Webseiten eng zusammen. Besuchen Sie Google für weitere Informationen
